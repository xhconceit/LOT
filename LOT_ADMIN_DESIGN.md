# LOT 后台管理系统 - Vercel 风格暗色主题设计

## 设计原则

### 1. 极致对比

- 深黑背景与白色文字形成最高对比度
- 白色卡片在深黑背景上突出显示

### 2. 微妙细节

- 30% 不透明度的分隔线
- 15% 不透明度的卡片边框
- 微妙的阴影效果（3% 不透明度，白色）
- 激活导航项的白色阴影

### 3. 一致的间距系统

- 24/32/48px 的内边距层次
- 16/24/28px 的元素间距
- 统一 20px/12px/6px 的间距组

### 4. 优雅的圆角

- 12px 卡片圆角（克制且现代）
- 6px 导航激活项圆角（与卡片一致）

### 5. 品牌色应用

- 橙色 #F97316 作为唯一强调色
- 使用于：激活导航背景、Logo 分隔线、头像、按钮、趋势图标、搜索框下划线

### 6. 层次化文字

- Logo/标题：米黄色，24px，600 字重（最高层级）
- 卡片标签：浅灰褐，13px，500 字重（次层级）
- 卡片数值：米黄色，40px，700 字重，负字间距（核心数据）
- 导航文字：深灰/白色，14px（次层级）
- 占位符文字：浅灰，14px（辅助）

---

## 设计优势

1. **极致对比度**：深色背景 + 白色元素，最佳可读性
2. **专业感**：克制的 12px 圆角，不过度装饰
3. **高级质感**：微妙的阴影和边框，营造层次
4. **舒适间距**：充足的留白，避免拥挤
5. **品牌一致性**：橙色作为唯一强调色，不过度使用
6. **暗色护眼**：深黑主题，减少视觉疲劳
